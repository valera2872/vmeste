# Версия 0.2.1: человеческий язык интерфейса

Первая цепочка приложения переписана вокруг исходной боли пользователя: важную цель бывает трудно довести до результата в одиночку.

Правила текстов:

- говорить о конкретной ситуации и действии;
- не обвинять пользователя в недостатке силы воли;
- сразу объяснять доступную помощь;
- не использовать внутреннюю продуктовую терминологию без объяснения;
- не превращать приложение в обычный трекер задач.
