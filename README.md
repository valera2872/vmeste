# Вместе к цели

Первая техническая версия приложения, которое помогает человеку перейти от намерения к реальному действию.

## Ядро продукта

1. Пользователь называет конкретное дело.
2. Выбирает реалистичную продолжительность.
3. Начинает самостоятельно или с цифровым компаньоном.
4. При затруднении уменьшает шаг, а не отказывается от цели.
5. После завершения фиксирует реальный результат.
6. После пропуска приложение помогает вернуться, а не обнуляет прогресс.

Приложение предназначено для разных возрастов. Возрастная группа используется только для адаптации языка, примеров и социальных настроек, а не для запрета доступа.

## Что уже реализовано в прототипе 0.1

- возрастно-адаптивный первый запуск;
- главный экран с быстрым стартом;
- создание одной активной цели;
- самостоятельная сессия;
- цифровой компаньон Lite;
- таймер;
- режим минимального шага;
- помощь «Застрял»;
- фиксация результата;
- повтор успешной сессии;
- история последних действий;
- мягкое возвращение после неудачной сессии.

## Следующая версия 0.2

- приглашение знакомого человека по ссылке или коду;
- гостевой вход без обязательной регистрации;
- статусы «готов / начал / завершил»;
- совместный старт;
- страховка от неявки партнёра;
- реакция «Ты помог мне начать»;
- повторные пары и избранные партнёры.

## Запуск

Репозиторий содержит Flutter-код приложения. Платформенные каталоги `android` и `ios` будут добавлены отдельным техническим шагом.

После установки Flutter:

```bash
flutter create . --project-name vmeste_goal --platforms=android,ios
flutter pub get
flutter run
```

Перед выполнением `flutter create .` сохраните текущий `lib/main.dart`, если используемая версия Flutter предлагает его заменить.

Подробные границы первой версии находятся в [docs/MVP.md](docs/MVP.md).
